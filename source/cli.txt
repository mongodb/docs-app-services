Options
-------

.. list-table::
   :header-rows: 1
   :widths: 20 10 10 60

   * - Name
     - Type
     - Required
     - Description
   * - --profile
     - string
     - false
     - Specify your profile (Default value: "default") This value defaults to "default".
   * - --telemetry
     - String
     - false
     - Enable/Disable CLI usage tracking for your current profile (Default value: "on"; Allowed values: "on", "off")
   * - -o, --output-target
     - string
     - false
     - Write CLI output to the specified filepath
   * - -f, --output-format
     - String
     - false
     - Set the CLI output format (Default value: <blank>; Allowed values: <blank>, "json")
   * - --disable-colors
     - 
     - false
     - Disable all CLI output styling (e.g. colors, font styles, etc.)
   * - -y, --yes
     - 
     - false
     - Automatically proceed through CLI commands by agreeing to any required user prompts
   * - -h, --help
     - 
     - false
     - help for app-services-cli

Related Commands
----------------

* :ref:`app-services-cli-function` - Interact with the Functions of your app (alias: functions)
* :ref:`app-services-cli-login` - Log the CLI into App Services using a MongoDB Cloud API Key
* :ref:`app-services-cli-logout` - Log the CLI out of App Services
* :ref:`app-services-cli-logs` - Interact with the Logs of your app (alias: log)
* :ref:`app-services-cli-pull` - Exports the latest version of your app into your local directory (alias: export)
* :ref:`app-services-cli-push` - Imports and deploys changes from your local directory to your app (alias: import)
* :ref:`app-services-cli-schema` - Manage the Schemas of your app (alias: schemas)
* :ref:`app-services-cli-secrets` - Manage the Secrets of your app (alias: secret)
* :ref:`app-services-cli-users` - Manage the Users of your app (alias: user)
* :ref:`app-services-cli-whoami` - Display information about the current user

================
Values & Secrets
================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 2
   :class: singlecol

As part of the deprecation of Atlas App Services, Values and Secrets will only
be available through Triggers. The Values and Secrets documentation has been
moved to `MongoDB Atlas Documentation -> Triggers/Functions
<https://www.mongodb.com/docs/atlas/atlas-ui/triggers/functions/>`__.

For more details, refer to :ref:`app-services-deprecation`.
